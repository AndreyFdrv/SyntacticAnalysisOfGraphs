Требование к ОС: Ubuntu

Сборка:
1. Запустить терминал
2. Перейти в папку SyntacticAnalysisOfGraphs
3. Запустить скрипт build, введя ./build

Запуск:
1. Запустить терминал
2. Перейти в папку SyntacticAnalysisOfGraphs
3. Ввести в терминале ./SyntacticAnalysis путь_к_файлу_с_грамматикой 
	путь_к_файлу_с_графом, например ./SyntacticAnalysis ./SyntacticAnalysis ./tests/Grammar1.txt ./tests/Graph1.dot

Тесты:
Тесты расположены в папке tests
